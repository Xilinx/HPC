*************************
L1 Primitives User Guide
*************************

L1 primitives provide the hardware building blocks for implementing RTM imaging and high precision ML inferences. 
They are implemented as C++ template classes and functions. Users can find these implementations in the ``L1/include/hw`` 
directory of the Vitis hpc library.
