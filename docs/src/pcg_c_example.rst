=========================
C Example using Makefile
=========================

* Open a terminal using your Linux credentials
* Execute following commands to run the example

.. code-block:: bash

   mkdir pcg-example
   cd pcg-example 
   cp /opt/xilinx/apps/hpc/pcg/1.0/examples.zip .
   unzip examples.zip
   cd examples/c
   make
   make run
